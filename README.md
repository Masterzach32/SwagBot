# SwagBot
Very basic music and moderation bot for discord

Add the bot to your discord server:
https://discordapp.com/oauth2/authorize?client_id=217065780078968833&scope=bot&permissions=8

Dependencies:
- ffmpeg
- ffprobe
- youtube-dl
- Discord4J
